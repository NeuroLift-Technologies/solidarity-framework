# Quick Start TOI Template

**New to OTOI?** This simplified template helps you create a basic Personal Terms of Interaction in just a few minutes. You can always expand it later!

## Essential Preferences

### Communication Style
**How should AI talk to me?**
- [ ] Keep it professional and clear
- [ ] Be casual and friendly  
- [ ] Adapt to the situation

### Information Processing  
**How do I work best with information?**
- [ ] Give me bullet points and lists
- [ ] Use regular paragraphs
- [ ] Show me step-by-step instructions
- [ ] Let me ask for details when I need them

### Privacy Basics
**How should AI handle my data?**
- [ ] Delete everything when we're done talking
- [ ] Keep it for a while then delete
- [ ] I'll tell you when to delete
- [ ] Standard privacy practices are fine

### Energy & Interaction
**How should we pace our interactions?**
- [ ] I need time to process - don't rush
- [ ] Normal conversation pace is fine
- [ ] I like quick, efficient interactions
- [ ] Adapt based on how I'm responding

## Optional Additions

### If You're Neurodivergent
- [ ] I prefer very direct communication
- [ ] I need consistent interaction patterns  
- [ ] Please avoid overwhelming amounts of information
- [ ] I work better with visual structure (headings, spacing)
- [ ] I need reminders or check-ins
- [ ] Sensory considerations matter to me

### If You Use Accessibility Tools
- [ ] I use a screen reader
- [ ] I navigate with keyboard only
- [ ] I need high contrast or large text
- [ ] I prefer simple, clear language
- [ ] Audio alternatives are helpful

### If You're Working in Teams
- [ ] I want my individual preferences respected in group settings
- [ ] Group consensus is usually fine
- [ ] I prefer to make final decisions myself
- [ ] I'm comfortable with AI coordination
